## Desenvolvimento

    git clone --depth=1  git@github.com:gpupo/cnova-sdk.git
    cd cnova-sdk;
    ant;

Personalize a configuração do ``phpunit``:

    cp phpunit.xml.dist phpunit.xml;

Personalize os parâmetros!



*Dica*: Verifique os logs gerados em ``var/log/main.log``
